## О проекте

**Plyr Video Player** — это простой, лёгкий, доступный и настраиваемый медиаплеер для HTML5, YouTube, Vimeo и российских видеохостингов (Rutube, VK, Yandex Cloud Video, Mail.ru). Проект поддерживает все современные браузеры и предоставляет богатый API для управления воспроизведением, субтитрами, качеством видео и полноэкранным режимом.

Плеер написан на «чистом» JavaScript (ES6) без зависимости от фреймворков, использует семантическую HTML-разметку и Sass для стилизации. Архитектура построена на принципах прогрессивного улучшения — базовое `<video>` или `<audio>` элемент работает без JavaScript, а Plyr добавляет расширенный функционал при его наличии.

## Ключевые возможности

| # | Возможность | Описание |
|---|-------------|----------|
| 1 | **HTML5 Video & Audio** | Нативная поддержка видео и аудио с кастомными контролами |
| 2 | **YouTube & Vimeo** | Интеграция с популярными видеохостингами через единый API |
| 3 | **Российские хостинги** | Поддержка Rutube, VK Video, Yandex Cloud Video, Mail.ru Video |

| 7 | **Полноэкранный режим** | Нативный fullscreen с fallback на «полное окно» |
| 8 | **Клавиатурные сокращения** | Управление воспроизведением через горячие клавиши |
| 9 | **Picture-in-Picture** | Поддержка режима «картинка в картинке» |
| 10 | **Скорость воспроизведения** | Настройка скорости от 0.5x до 4x |

| 12 | **Монетизация (Ads)** | Интеграция с VAST-рекламой через vi.ai |
| 13 | **Стриминг (HLS/DASH)** | Поддержка hls.js, Shaka Player и dash.js |
| 14 | **i18n** | Интернационализация интерфейса (перевод контролов) |
| 15 | **API** | Полный программный API: play, pause, seek, volume, fullscreen |
| 16 | **События** | Унифицированные события для всех типов медиа |
| 17 | **Адаптивный дизайн** | Responsive-вёрстка для любых размеров экрана |
| 18 | **Доступность** | Полная поддержка screen readers и ARIA-атрибутов |
| 19 | **Sass** | Кастомизация стилей через Sass-переменные и CSS Custom Properties |
| 20 | **Browsersync UI** | Встроенный сервер разработки с live reload и ghost mode |

## Установка и запуск

### Предварительные требования

- **Node.js** версии 18 или выше
- **npm** в качестве пакетного менеджера

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/QuadDarv1ne/plyr-video.git
cd plyr-video

# Установить зависимости
npm install

# Запустить в режиме разработки
npm start
```

Приложение будет доступно по адресу [http://localhost:3007](http://localhost:3007)
Панель Browsersync UI: [http://localhost:3008](http://localhost:3008)

### Сборка для продакшена

```bash
# Сборка проекта
npm run build

# Запуск тестов
npm test
```
