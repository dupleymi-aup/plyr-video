- Government Decree No. 1119 dated 01.11.2012 "On Approval of Requirements

- Civil Code of the Russian Federation;
- Federal Law No. 149-FZ dated 27.07.2006 "On Information, Information

### 2. DATA PROCESSING OPERATOR

2.1. Operator: **Dupley Maxim Igorevich**

2.2. Software repository: https://github.com/QuadDarv1ne/plyr-video

2.3. The Operator does not independently collect, store, or process Users'
personal data when using the Software as a standalone software product

### 3. NATURE OF PERSONAL DATA

3.1. The Software is a client-side JavaScript library for audio and video
playback in a web browser and does not involve independent collection of
personal data by the Operator.

3.2. Personal data may be processed by the User (the party integrating the
Software into their product or service) under the following circumstances:

a) when using captions (WebVTT) — data about language and display preferences;

b) when using transcription — text data related to media file content;

c) when using settings persistence (localStorage) — user preferences for
   volume, playback speed, caption language.

3.3. The Software does not transmit personal data to the Operator. All data
is processed exclusively on the client side (in the User's browser).

### 4. PURPOSES OF PERSONAL DATA PROCESSING

4.1. Where the User processes personal data using the Software, the purposes
are:

a) ensuring Software functionality according to User preferences;

b) persisting user settings (volume, playback speed, caption language)
   between sessions;

c) ensuring accessibility for persons with disabilities;

d) improving content playback quality.

### 5. LEGAL GROUNDS FOR PROCESSING

5.1. Legal grounds for personal data processing include:

a) consent of the personal data subject (Part 1, Article 6, Federal Law
   No. 152-FZ);

b) necessity for performance of a contract (Part 1, Article 6, Federal Law
   No. 152-FZ);

c) legitimate interests of the operator, provided this does not violate the
   rights and freedoms of the data subject (Part 1, Article 6, Federal Law
   No. 152-FZ).

### 6. VOLUME AND CATEGORIES OF PERSONAL DATA

6.1. The following categories of data may be processed:

a) **Technical data:** browser type/version, device type and OS, screen
   resolution, network connection parameters.

b) **Preference data:** volume settings, playback speed, interface and
   caption language, video quality settings.

c) **Usage data:** playback history (when using localStorage), playback
   position when saving progress.

6.2. The Software **does not process** special categories of personal data
(Article 10, Federal Law No. 152-FZ), including data on racial origin,
nationality, political views, religious or philosophical beliefs, health,
or intimate life.

6.3. The Software **does not process** biometric personal data (Article 11,
Federal Law No. 152-FZ).

### 7. METHODS AND CONDITIONS OF PROCESSING

7.1. Personal data processing methods include:

a) **localStorage / sessionStorage** — for persisting user settings on the
   client side. Data is not transmitted to the Operator's server.

b) **Cookies** — may be set by third-party services (YouTube, Vimeo, Rutube,
   VK Video, etc.) when using corresponding plugins. See the Cookie Policy

c) **API requests** — when using video hosting plugins, data is transmitted
