### 2. ЧТО ТАКОЕ ФАЙЛЫ COOKIE

2.1. **Файлы cookie** — это небольшие текстовые файлы, которые сохраняются
на устройстве Пользователя (компьютере, планшете, смартфоне) при посещении
веб-сайтов и использовании веб-приложений.

2.2. Файлы cookie используются для:

- обеспечения корректной работы веб-сервисов;
- сохранения пользовательских предпочтений и настроек;
- анализа использования сервисов;
- обеспечения безопасности.

2.3. Помимо файлов cookie, для хранения данных могут использоваться
аналогичные технологии: **localStorage**, **sessionStorage**, **IndexedDB**,
**Service Workers**.

### 3. ФАЙЛЫ COOKIE, УСТАНАВЛИВАЕМЫЕ ПРОГРАММНЫМ ОБЕСПЕЧЕНИЕМ

3.1. Программное обеспечение «Plyr Video Player» **не устанавливает файлы
cookie** самостоятельно.

3.2. Программное обеспечение использует **localStorage** браузера для
сохранения следующих данных:

| Данные | Ключ localStorage | Цель | Срок хранения |
|--------|-------------------|------|---------------|
| Громкость | `plyr-volume` | Сохранение уровня громкости между сеансами | До удаления Пользователем |
| Скорость воспроизведения | `plyr-speed` | Сохранение выбранной скорости | До удаления Пользователем |
| Настройки плеера | `plyr-settings` | Сохранение общих настроек (язык, качество) | До удаления Пользователем |
| Прогресс воспроизведения | `plyr-watched:*` | Сохранение позиции просмотра | До удаления Пользователем |

3.3. Данные в localStorage:

- **не передаются** на сервер Правообладателя;
- обрабатываются исключительно на стороне клиента (в браузере);
- могут быть удалены Пользователем в любой момент через настройки браузера;
- не содержат персональных данных в смысле Федерального закона № 152-ФЗ.

## ENGLISH VERSION

### 1. GENERAL PROVISIONS

1.1. This Cookie Policy defines the procedures for data storage technologies
on the User's device when using the "Plyr Video Player" software.

1.2. This Policy is developed in accordance with:

- Federal Law No. 152-FZ "On Personal Data";
- Federal Law No. 149-FZ "On Information, Information Technologies, and
  Protection of Information";
- Roskomnadzor requirements for cookie usage;
- international standards including the EU ePrivacy Directive and GDPR.

1.3. The copyright holder is **Dupley Maxim Igorevich**.

### 2. WHAT ARE COOKIES

2.1. **Cookies** are small text files stored on the User's device when visiting
websites and using web applications.

2.2. Cookies are used for: ensuring proper service functionality, saving user
preferences, analyzing service usage, and ensuring security.

2.3. In addition to cookies, similar technologies may be used: localStorage,
sessionStorage, IndexedDB, Service Workers.

### 3. COOKIES SET BY THE SOFTWARE

3.1. The "Plyr Video Player" software **does not set cookies** independently.

3.2. The software uses browser **localStorage** to store the following data:

| Data | localStorage Key | Purpose | Retention |
|------|-----------------|---------|-----------|
| Volume | `plyr-volume` | Persist volume level between sessions | Until deleted by User |
| Playback speed | `plyr-speed` | Persist selected speed | Until deleted by User |
| Player settings | `plyr-settings` | Persist general settings (language, quality) | Until deleted by User |
| Playback progress | `plyr-watched:*` | Save viewing position | Until deleted by User |

3.3. localStorage data is **not transmitted** to the Copyright Holder's server,
is processed exclusively on the client side, and can be deleted by the User
at any time through browser settings.

### 4. THIRD-PARTY COOKIES

4.1. When using integration plugins with video hosting services, cookies may
be set by those third parties.

4.2. **YouTube** (Google LLC): YSC (session identification), VISITOR_INFO1_LIVE
(bandwidth estimation), CONSENT (consent storage).
Privacy: https://policies.google.com/privacy

4.3. **Vimeo** (Vimeo Inc.): vuid (user identification), __utm* (analytics).
Privacy: https://vimeo.com/privacy

4.4. **Rutube** (JSC "Rutube"): sessionid, _ym_* (Yandex.Metrica analytics).
Privacy: https://rutube.ru/info/privacy/

4.5. **VK Video** (LLC "VKontakte"): remixsid, _ym_* (Yandex.Metrica).
Privacy: https://vk.com/privacy

4.6. **Yandex Cloud Video** (LLC "Yandex"): yandexuid, _ym_*.
Privacy: https://yandex.ru/legal/confidential/

4.7. **Mail.ru Video** (LLC "VK"): mrcu, _ym_*.
Privacy: https://help.mail.ru/privacy
